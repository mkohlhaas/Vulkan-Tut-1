# Vulkan Tutorial

## 1. Window

- Add [GLFW](https://www.glfw.org/) as a submodule:
```shell
git submodule add https://github.com/glfw/glfw.git
git checkout -b 3.4
git config submodule.glfw.ignore all
git config --list --local
```
